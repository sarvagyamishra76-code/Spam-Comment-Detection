# Spam Comment Detection Using NLP and Machine Learning

## Project Overview

This project detects whether a comment or message is Spam or Ham using Natural Language Processing (NLP) and Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* NLTK
* Scikit-learn
* Streamlit
* Tableau
* GitHub

## Workflow

Dataset Collection → Data Cleaning → NLP Preprocessing → EDA → TF-IDF Vectorization → Model Training → Model Evaluation → Streamlit Deployment
